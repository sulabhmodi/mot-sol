FIO Test Scenarios

Sequential Write Test: Tests large block write throughput.



fio --name=seqwrite \
    --directory=/data \
    --size=1G \
    --bs=1M \
    --rw=write \
    --ioengine=libaio \
    --direct=1 \
    --numjobs=1 \
    --runtime=60 \
    --group_reporting
